Privacy Policy

App Name: BIM Sheet Manager

Publisher: Khaled Saieed Studio

Last Updated: March 2026

1. Introduction

Khaled Saieed Studio is committed to protecting the privacy of users of the BIM Sheet Manager add-in for Autodesk Revit. This Privacy Policy explains what data we collect, how we use it, and your rights as a user.

2. Data Collection

The BIM Sheet Manager add-in does not collect, store, transmit, or share any personal data or user information. All operations are executed locally on your machine within the Autodesk Revit environment. No project data, sheet names, or file contents are ever transmitted outside of Revit.

3. Local File Access

The add-in reads Excel (.xlsx) files that you explicitly select via a file dialog, and writes exported Excel files to a path you choose on your own machine. These files are processed entirely locally and are never uploaded or transmitted to any external server.

4. Third-Party Services

This add-in does not integrate with any third-party analytics, advertising, or data collection services.

5. Internet Access

The BIM Sheet Manager add-in does not require or use an internet connection. No data is sent to external servers.

6. Changes to This Policy

We reserve the right to update this Privacy Policy at any time. Any changes will be reflected on this page with an updated date.
